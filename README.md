# FinTrack - Personal Finance Tracker

A full-stack personal finance tracking web application with AI-powered insights and recommendations.

## Features
- User registration and login
- Income tracking by source
- Expense tracking by category
- Monthly budget management
- Real-time dashboard with charts
- AI-generated financial insights including:
  - Budget alerts
  - Month-over-month spending analysis
  - Savings rate advice
  - Spending predictions

## Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | HTML, CSS, JavaScript |
| Backend | Python, Flask |
| Database | MySQL |
| AI Module | Python (rule-based + scikit-learn) |
| Charts | Chart.js |
